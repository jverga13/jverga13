## 🔬 Featured Engineering Projects & Experience

- 🤖 **Atombot Swarm Robotics Lab (Research Lead)**  
  Architected a deterministic **500 kbps**, thread-safe **CAN pipeline** on dual-core **RP2350B ARM firmware**. Integrated **4D LiDAR** and **Intel RealSense** with **ROS2 + C++** for SLAM-focused navigation.

- 🕹️ **Whack-A-Math (FPGA Educational Game)**  
  Engineered a real-time interactive game on **Altera DE2-115 FPGA** using **SystemVerilog** hardware logic and **Assembly** low-level control. Built custom drivers for resistive touchscreen, audio codec, and SD card with an **FSM-based architecture**.

- ⚛️ **Quantum ML Play Prediction Model**  
  Developed a **Quantum Kernel SVM (QSVM)** with **PennyLane + Qiskit**, achieving **74.67% test accuracy** and winning the **Observer's Choice Award** at the IEEE ISIT 2025 Quantum Hackathon. Co-authored a forthcoming summary article in **IEEE BITS Magazine**.

- 💻 **MGoMom Logistical Platform**  
  Designed a centralized full-stack digital ordering platform in **JavaScript (ES6+) + Firebase**, including an Admin Dashboard powered by Firestore real-time **onSnapshot** state management.

- 🚀 **Michigan Mars Rover Team (MRover)**  
  Designed high-reliability power distribution **PCBs in Altium Designer**, optimized for thermal management and signal integrity in high-vibration, space-grade subsystems.
